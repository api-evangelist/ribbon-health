# Ribbon Health (ribbon-health)

Ribbon Health (now part of H1) is a healthcare provider data platform offering a REST API for searching provider directories, verifying insurance networks, checking provider availability, and enriching clinical data. The H1 API provides comprehensive data on doctors, insurance plans, procedures, costs, and quality scores across the United States, enabling health plans, providers, and digital health solutions to build accurate provider directories and care navigation tools.

## APIs

### H1 API (Ribbon Health)

The H1 REST API provides healthcare enterprises with accurate data on providers, insurance plans, costs, and quality of care. Key capabilities include provider search and lookup by NPI, location search, insurance network verification, eligibility checking, procedure cost estimation, and custom provider directory management.

- Documentation: https://ribbon.readme.io/docs/welcome-to-the-ribbon-health-api
- Base URL: https://api.ribbonhealth.com/v1
- API Version: 2.2
